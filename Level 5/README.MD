## **1Ô∏è‚É£ Classes e Objetos**  

### **üìå Definindo uma Classe**  
```python
class Carro:
    # Atributo de classe (compartilhado entre inst√¢ncias)
    rodas = 4  

    # M√©todo construtor (__init__)
    def __init__(self, marca, modelo, ano):
        self.marca = marca    # Atributo de inst√¢ncia
        self.modelo = modelo
```

## **2Ô∏è‚É£ Heran√ßa e Polimorfismo**  

### **üìå Heran√ßa Simples**  
```python
class Veiculo:
    def __init__(self, marca, modelo):
        self.marca = marca
        self.modelo = modelo

    def acelerar(self):
        print("Acelerando...")

class Carro(Veiculo):  # Herda de Veiculo
    def __init__(self, marca, modelo, portas):
        super().__init__(marca, modelo)  # Chama o construtor da classe pai
        self.portas = portas

    # Sobrescreve o m√©todo acelerar (Polimorfismo)
    def acelerar(self):
        print(f"{self.modelo} acelerando r√°pido!")

carro = Carro("Ford", "Mustang", 2)
carro.acelerar()  # Sa√≠da: "Mustang acelerando r√°pido!"
```

## **3Ô∏è‚É£ Encapsulamento (Atributos Privados)**  

### **üìå Atributos Privados (`_` e `__`)**  
```python
class ContaBancaria:
    def __init__(self, saldo):
        self.__saldo = saldo  # Atributo privado (__)

    def depositar(self, valor):
        if valor > 0:
            self.__saldo += valor
            print(f"Dep√≥sito de R${valor} realizado.")

    def ver_saldo(self):
        return self.__saldo

conta = ContaBancaria(1000)
conta.depositar(500)
print(conta.ver_saldo())  # 1500
# print(conta.__saldo)  # Erro! Atributo privado.
```

## **4Ô∏è‚É£ M√©todos Especiais (Dunder Methods)**  

### **üìå `__str__` (Representa√ß√£o em String)**  
```python
class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def __str__(self):
        return f"{self.nome}, {self.idade} anos"

```

### **üìå `__eq__` (Compara√ß√£o de Objetos)**  
```python
class Ponto:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __eq__(self, outro):
        return self.x == outro.x and self.y == outro.y

p1 = Ponto(1, 2)
p2 = Ponto(1, 2)
print(p1 == p2)  # True (sem __eq__, seria False)
```

## **5Ô∏è‚É£ Classes Abstratas (`ABC`)**  

### **üìå Definindo uma Classe Abstrata**  
```python
from abc import ABC, abstractmethod

class Animal(ABC):  # Classe abstrata
    @abstractmethod
    def emitir_som(self):
        pass

class Cachorro(Animal):
    def emitir_som(self):
        print("Au au!")

class Gato(Animal):
    def emitir_som(self):
        print("Miau!")

# animal = Animal()  # Erro! N√£o pode instanciar classe abstrata.
cachorro = Cachorro()
cachorro.emitir_som()  # Au au!
```

## **üìö Exemplo Completo**  

```python
class Produto:
    def __init__(self, nome, preco):
        self.nome = nome
        self.preco = preco

    def desconto(self, percentual):

    def __init__(self, nome, preco, autor):
        super().__init__(nome, preco)
        self.autor = autor

    def __str__(self):
        return f"Livro: {self.nome} ({self.autor}), R${self.preco}"
```
