# **Convers√£o de Tipos em Python**  

Este guia explica como **alterar o tipo de uma vari√°vel** em Python, incluindo m√©todos nativos, boas pr√°ticas e exemplos pr√°ticos.

## **üìå Tipos de Dados em Python**  
Python possui v√°rios tipos de dados b√°sicos:  

| **Tipo**      | **Descri√ß√£o**           | **Exemplo**       |
|--------------|------------------------|------------------|
| `int`        | N√∫meros inteiros        | `10`, `-5`       |
| `float`      | N√∫meros decimais        | `3.14`, `-0.5`   |
| `str`        | Texto (strings)         | `"Python"`       |
| `bool`       | Valores l√≥gicos         | `True`, `False`  |
| `list`       | Listas mut√°veis         | `[1, 2, 3]`      |
| `tuple`      | Tuplas imut√°veis        | `(1, 2, 3)`      |
| `dict`       | Dicion√°rios (chave-valor) | `{"nome": "Jo√£o"}` |

## **üîÑ Como Converter Tipos?**  

### **1Ô∏è‚É£ `int()` ‚Üí Converte para inteiro**  
```python

# Cuidado: Se n√£o for um n√∫mero v√°lido, causa erro!
# int("dez") ‚Üí ValueError
```

### **2Ô∏è‚É£ `float()` ‚Üí Converte para decimal**  
```python

numero_float = float(numero_str)  # Converte str para float
print(numero_float, type(numero_float))  # Sa√≠da: 3.14 <class 'float'>
```

### **3Ô∏è‚É£ `str()` ‚Üí Converte para string**  
```python

texto = str(numero)  # Converte int para str
print(texto, type(texto))  # Sa√≠da: "42" <class 'str'>
```

### **4Ô∏è‚É£ `bool()` ‚Üí Converte para booleano**  
```python

valor_bool = bool(valor)  # Converte para True/False
print(valor_bool, type(valor_bool))  # Sa√≠da: True <class 'bool'>

# Valores que viram False:
# 0, "", [], None, False
```

### **5Ô∏è‚É£ `list()`, `tuple()`, `dict()` ‚Üí Convers√£o entre cole√ß√µes**  
```python

```

## **‚ö†Ô∏è Cuidados na Convers√£o**  

### **1. Convers√µes inv√°lidas causam `ValueError`**  
```python
# int("abc") ‚Üí ValueError
```

### **2. Perda de precis√£o em `int` ‚Üí `float`**  

### **3. Convers√£o impl√≠cita em opera√ß√µes**

## **üìö Resumo das Fun√ß√µes de Convers√£o**  

| **Fun√ß√£o** | **Converte para** | **Exemplo** |
|------------|------------------|-------------|
| `int()`    | Inteiro          | `int("10") ‚Üí 10` |
| `float()`  | Decimal          | `float("3.14") ‚Üí 3.14` |

## **üöÄ Conclus√£o**

- Sempre valide entradas antes de converter.  
- Cuidado com perda de dados em convers√µes (`float` ‚Üí `int`).
