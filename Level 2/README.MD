## üìå **Loops (`for`, `while`)**

### 1. **Loop `for`**
#### Itera√ß√£o em Sequ√™ncias:
```python

    print(fruta.upper())

    print(letra)

# Range (gera√ß√£o de n√∫meros)

```

#### Com `enumerate()` (√≠ndice + valor):

### 2. **Loop `while`**
#### Executa enquanto a condi√ß√£o for verdadeira:
```python

    print(f"Contagem: {contador}")
    contador += 1  # Incremento para evitar loop infinito
```

#### Exemplo com `break`:
```python

    if comando.lower() == "sair":
        break  # Sai do loop
    print("Comando inv√°lido!")
```

### 3. **Controle de Loops**
| Comando   | Efeito                              | Exemplo                     |
|-----------|-------------------------------------|-----------------------------|
| `break`   | Interrompe o loop imediatamente     | `if erro: break`            |
| `continue`| Pula para a pr√≥xima itera√ß√£o        | `if x % 2 == 0: continue`   |
| `else`    | Executa se o loop **n√£o** for interrompido por `break` | `for x in lista: ... else: print("Fim!")` |
